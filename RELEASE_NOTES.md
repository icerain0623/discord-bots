# Release Notes

## v0.7.0 — 匿名コンタクト機能（🧪 テスト版）

> 2026-03-21

### 追加（🧪 テスト段階）

- `/contact` コマンド — モデレーターに匿名で通報・相談を送信
  - モーダルで本文を入力、モデレーターチャンネルに匿名Embedとして投稿
  - モデレーターが「返信する」ボタンで匿名のまま返信可能（DM経由）
  - 送信者もDM内の「返信する」ボタンで追加メッセージ可能（複数往復対応）
  - データは30日間KVに保持（やり取りがあるたびにTTLを延長）
- `CONTACT_CHANNEL_ID` 環境変数 — モデレーター用チャンネルの指定

---

## v0.7.0 — 交流マッチング機能（テスト機能）

> 2026-03-21

### 追加

- `/matchup` コマンド — メンバーをランダムにグループ分けして専用チャンネルで交流（🧪 テスト機能）
  - `/matchup start group_size:N` — 募集開始（参加ボタン付きメッセージを投稿）
  - `/matchup run` — マッチング実行（ランダムグループ分け + プライベートチャンネル作成）
  - `/matchup terminate` — イベント終了（チャンネル削除 + クリーンアップ）
  - `/matchup topics add/remove/list` — トピック管理
- 参加フロー: ボタンクリック → トピック選択（セレクトメニュー） → 自由トピック入力（任意モーダル）
- グループの共通トピックを自動提示して会話のきっかけに
- `MATCHUP_KV` namespace を追加（ギルドスコープのキー設計）

### 技術的変更

- Worker fetch ハンドラーに `ctx` パラメータを追加（`ctx.waitUntil()` で非同期処理）
- `discordApi.js` にチャンネル作成・削除・カテゴリ作成・メッセージ編集・フォローアップ送信を追加
- `postMessage` が文字列と embed オブジェクトの両方を受け付けるように変更

## v0.5.0 — 絵文字ランキング: バッチ集計 + 期間別表示

> 2026-03-20

### 変更

- 絵文字集計をリアルタイム（Worker 内）からローカル PC バッチ集計 + KV 読み取りに変更
  - Cloudflare Workers Free プランの30秒制限を回避
  - チャンネル数・スレッド数の上限なく全件集計可能に
- テキストチャンネルとフォーラムスレッドを合算して1つの KV に保存
- `/emoji-stats` コマンドに期間オプションを追加（今週 / 先週 / 今月 / 先月 / 全期間）
- `対象` オプション（チャンネル / フォーラム選択）を廃止

### 追加

- `npm run collect` — ローカルバッチ集計スクリプト
  - 初回は全メッセージ取得、2回目以降は差分取得
  - 週単位（ISO 8601、月曜始まり）でデータを蓄積
- `src/utils/weekUtils.js` — ISO 週番号・期間フィルタユーティリティ

## v0.1.0 — 初回リリース / Cloudflare Workers 移行

> 2026-03-18

### 機能

- `/setup-intro` コマンドでチャンネルに自己紹介パネルを設置
- 3〜4ステップのモーダルで自己紹介を入力し、指定チャンネルに投稿
- 全ステップがエフェメラル（本人のみ表示）
- いつでもキャンセル可能、セッションは30分で自動破棄
- 未入力フィールドは「未回答」として投稿

### アーキテクチャ

- discord.js + Node.js から **Cloudflare Workers** に移行
- セッション管理を Map から **Workers KV** に変更
- Discord HTTP Interactions（Webhook）方式に対応
- Ed25519 署名検証を実装
