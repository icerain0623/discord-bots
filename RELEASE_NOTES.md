# Release Notes

## v0.3.0 — 自己紹介「もっと！」セクション復活（任意回答式）

> 2026-03-19 | [PR #3](https://github.com/icerain0623/discord-bots/pull/3)

### 追加

- ステップ 3/3 完了後に「もっと回答する ➕」ボタンを表示
- 7つの追加質問を2つのモーダルで任意回答できるように
  - いま欲しいもの / ペットを飼うなら / 好きなブランド / 休日はどう過ごす？ / 返信は早い？ / きのこ派 or たけのこ派 / たい焼きの食べ方
- 追加質問に1つでも回答がある場合のみ、投稿に【もっと！】セクションを表示
- 回答しない場合は「確認へ進む →」でスキップ可能

## v0.1.0 — 初回リリース / Cloudflare Workers 移行

> 2026-03-18

### 機能

- `/setup-intro` コマンドでチャンネルに自己紹介パネルを設置
- 3〜4ステップのモーダルで自己紹介を入力し、指定チャンネルに投稿
- 全ステップがエフェメラル（本人のみ表示）
- いつでもキャンセル可能、セッションは30分で自動破棄
- 未入力フィールドは「未回答」として投稿

### アーキテクチャ

- discord.js + Node.js から **Cloudflare Workers** に移行
- セッション管理を Map から **Workers KV** に変更
- Discord HTTP Interactions（Webhook）方式に対応
- Ed25519 署名検証を実装

# Roadmap

## 自己紹介機能の強化

- [ ] **画像生成** — 自己紹介シートの写真にテキストを合成して投稿

## 川柳検知 Bot（新機能）

メッセージが 5・7・5 のリズムになっている投稿を自動検知して反応する機能。

- [ ] スペース区切りで 5・7・5 に分かれている投稿のみを川柳として検知
- [ ] 同一ユーザーからの検知は **1日1回まで**
- [ ] 川柳検知の間隔を **最低10分** 空ける（連続反応を防止）

## 月間ランキング（新機能）

サーバー内のアクティビティを集計し、月間ランキングを生成する機能。

- [ ] **絵文字ランキング** — 全絵文字の使用回数を記録・集計
- [ ] **メンションランキング**（実装未定） — ユーザーごとのメンション回数を記録
