# ユーザーテストガイド

一般メンバー向けに、各コマンドが正しく動作するかをテストする手順書です。
権限によって使えるコマンド・使えないコマンドがあるので、両方を確認してください。

---

## テスト前の準備

- テスト用のDiscordアカウントを用意する（または一般権限のロールに切り替える）
- テストアカウントには **管理者権限・サーバー管理権限・メッセージ管理権限をいずれも付与しない**
- Botがオンラインであることを確認する

---

### 3. 検閲モード別テスト

`検閲`コマンドは `/censor-settings` の設定値によって一般ユーザーの挙動が変わります。
**管理者側で設定を切り替えながら**テストしてください。

#### モード: `all`（全メッセージ対象 — デフォルト）

| # | 手順 | 期待される結果 | OK/NG |
|---|------|--------------|:-----:|
| 8-1 | 管理者が `/censor-settings mode:all` を実行する | 設定完了メッセージが表示される | |
| 8-2 | 一般ユーザーがメッセージ管理権限を持っていない状態で `検閲` を使う | 権限エラーが表示される | |
| 8-3 | 一般ユーザーにメッセージ管理権限を付与して `検閲` を使う | メッセージが検閲される（██████ に置き換わる） | |

#### モード: `self`（自分のメッセージのみ）

| # | 手順 | 期待される結果 | OK/NG |
|---|------|--------------|:-----:|
| 9-1 | 管理者が `/censor-settings mode:self` を実行する | 設定完了メッセージが表示される | |
| 9-2 | 一般ユーザーが **自分の** メッセージに `検閲` を使う | メッセージが検閲される | |
| 9-3 | 一般ユーザーが **他人の** メッセージに `検閲` を使う | エラーが表示される（自分のメッセージのみ対象） | |

#### モード: `off`（機能無効）

| # | 手順 | 期待される結果 | OK/NG |
|---|------|--------------|:-----:|
| 10-1 | 管理者が `/censor-settings mode:off` を実行する | 設定完了メッセージが表示される | |
| 10-2 | `検閲` を使う（誰でも） | 機能が無効化されている旨のメッセージが表示される | |

## 注意事項

- 権限エラーのメッセージは ephemeral（本人にのみ表示）なので、テスターのみが確認できます
- `/contact` のテストでは実際にモデレーターチャンネルに投稿されるため、テスト中である旨を事前に共有してください
- 検閲モード別テストでは管理者の協力が必要です
