# 1文リレー

参加者が一文ずつ順番に文章を追加していく共同創作イベント機能です。終了後に全文を匿名投稿し、執筆者のネタバレ開示もできます。

## コマンド

```bash
/relay start topic:<お題> first_sentence:<最初の一文>
```

リレーを開始します。チャンネルに参加ボタン付きパネルが投稿されます。

```bash
/relay status
```

現在の全文と各文の執筆者を自分だけに表示します。

```bash
/relay last
```

最後に追加された一文と執筆者を自分だけに表示します。

```bash
/relay delete number:<番号>
```

指定した番号の文を削除します。

```bash
/relay end
```

リレーを終了し、パネルの追記ボタンを無効化します（データは保持されます）。

```bash
/relay post channel:<チャンネル>
```

リレーの全文を匿名で指定チャンネルに投稿します（執筆者名は表示されません）。

```bash
/relay reveal channel:<チャンネル>
```

執筆者一覧（ネタバレ）を指定チャンネルに投稿します。

```bash
/relay terminate
```

リレーデータを完全に削除します。パネルメッセージも更新されます。

**必要権限:** サーバーの管理（全サブコマンド共通）

## フロー

```
/relay start でリレー開始・パネル投稿
    ↓
参加者がパネルのボタン「一文を追加する」をクリック
    ↓
モーダルで一文を入力して送信
    ↓
（繰り返し）
    ↓
/relay end でボタンを無効化
    ↓
/relay post で全文を匿名投稿
    ↓
/relay reveal でネタバレ投稿（任意）
    ↓
/relay terminate でデータ削除
```

## 注意

- 同時に開催できるリレーは1サーバーにつき1件です。
- `terminate` を実行するまでデータは保持されます。`end` だけでは削除されません。
- 全文が2000文字を超える場合、投稿は自動的に複数メッセージに分割されます。
