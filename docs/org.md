# 組織図自動編成

ロールと部門の設定をもとに、サーバーの組織図をチャンネルに自動生成・更新できる機能です。

## コマンド

```bash
/org config
```

部門・ロールのマッピング設定をモーダルで編集します。

```bash
/org dept-add name:<部門名>
```

部門を追加します。

```bash
/org dept-remove name:<部門名>
```

部門を削除します。

```bash
/org role-add dept:<部門名> role:<ロール名>
```

指定した部門にロールを紐付けます。

```bash
/org role-remove dept:<部門名> role:<ロール名>
```

指定した部門からロールの紐付けを解除します。

```bash
/org setup channel:<チャンネル>
```

指定チャンネルに組織図パネルを新規投稿します。既存のパネルがある場合は削除して再投稿します。

```bash
/org refresh
```

既存の組織図パネルを最新のロール・メンバー情報で更新します。

```bash
/org debug
```

組織図の内容を自分だけに表示してプレビューします（チャンネルには投稿しません）。

**必要権限:** サーバーの管理（全サブコマンド共通）

## フロー

```
/org dept-add で部門を作成
    ↓
/org role-add で各部門にロールを紐付け
    ↓
/org config でモーダルから設定を確認・調整（任意）
    ↓
/org setup channel:#組織図チャンネル でパネルを投稿
    ↓
メンバーやロールが変わったら /org refresh で更新
```

## 注意

- `setup` 前に必ず `/org dept-add` と `/org role-add` で部門・ロールを設定してください。
- `setup` を再実行すると旧パネルのメッセージが削除され、新しいパネルが投稿されます。
- `debug` はプレビュー専用のため、チャンネルへの投稿は行いません。
