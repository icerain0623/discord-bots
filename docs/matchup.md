# 交流マッチング

サーバーメンバーをランダムにグループ分けし、専用プライベートチャンネルで交流を促す機能です。

## フロー

```
管理者: /matchup topics add name:ゲーム   ← トピックを事前登録
管理者: /matchup start group_size:2       ← 募集開始（ボタン付きメッセージ投稿）
    ↓
メンバー: 「参加する」ボタンをクリック
    ↓ トピック選択（セレクトメニュー）
    ↓ 自由トピック入力（任意、モーダル）
    ↓
管理者: /matchup run                      ← マッチング実行・チャンネル作成
    ↓ ランダムにグループ分け
    ↓ グループごとにプライベートチャンネル作成
    ↓ 共通トピックを提示して会話のきっかけに
    ↓
管理者: /matchup terminate                ← チャンネル削除・イベント終了
```

## コマンド

| コマンド | 説明 |
|----------|------|
| `/matchup start group_size:N [category:]` | 募集を開始（グループサイズ 2〜4、カテゴリ省略時は自動作成） |
| `/matchup run` | マッチングを実行してチャンネルを作成 |
| `/matchup terminate` | イベントを終了（チャンネル削除） |
| `/matchup topics add name:` | トピックを追加（最大25個） |
| `/matchup topics remove name:` | トピックを削除 |
| `/matchup topics list` | 登録済みトピック一覧を表示 |

**必要権限:** サーバー管理（Manage Guild）

## 特徴

- Fisher-Yates シャッフルによるランダムなグループ分け
- グループメンバーの共通トピックを自動提示
- 専用カテゴリ内にプライベートチャンネルを作成（メンバー + Bot のみ閲覧可）
- 参加者が割り切れない場合は最後のグループに吸収
