# 検閲（ジョークコマンド）

メッセージを右クリック（Apps）から検閲できるコンテキストメニューコマンドです。

## 使い方

```
メッセージを右クリック → Apps → 検閲
```

**必要権限:** メッセージの管理（Manage Messages）を持つメンバーのみ

## 動作

```
対象メッセージを右クリック → Apps → 検閲
    ↓
元のメッセージが削除される
    ↓
Webhook で元の送信者の名前・アバターを使い
「██████████ [検閲済み]」を投稿
    ↓
Bot が1984風のセリフをランダムで投稿
```

## 管理者設定

```
/censor-settings mode:<モード>
```

**必要権限:** サーバー管理（Manage Guild）を持つメンバーのみ

| モード | 値 | 動作 |
|--------|-----|------|
| 全メッセージ対象 | `all` | 誰のメッセージでも検閲可能（デフォルト） |
| 自分のメッセージのみ | `self` | 自分が送信したメッセージだけ検閲可能 |
| オフ | `off` | 検閲機能を無効化 |

設定はサーバーごとに保存されます。

## 注意事項

- Bot のメッセージは検閲できません
- スレッド内でも動作します（Webhook は親チャンネルに作成）
- 削除後のメッセージは最下部に投稿されます（時系列の維持は不可）
- `ManageMessages` 権限のチェックはコマンド表示設定に関わらずサーバーサイドで行われます

## Bot に必要な権限

- メッセージの管理（Manage Messages）
- Webhook の管理（Manage Webhooks）
