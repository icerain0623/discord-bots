# じゃんけんPvP

肩書コイン経済の参加者同士でじゃんけん勝負ができる機能です。コインを賭けて1対1で対戦します。

## コマンド

```bash
/janken challenge user:<ユーザー> bet:<賭け金>
```

指定ユーザーにじゃんけん勝負を挑みます。チャンネルに挑戦メッセージが公開投稿され、相手が受諾・拒否を選びます。

**必要権限:** `/economy join` で参加済みであること（挑戦者・相手ともに）

## フロー

```
/janken challenge user:@相手 bet:100 で挑戦
    ↓
チャンネルに「受ける」「拒否する」ボタン付きメッセージが公開投稿
    ↓
相手が「受ける」をクリック → 両者の残高からコインがエスクロー
    ↓
挑戦者・相手それぞれにグー・チョキ・パーのボタンが表示
（手の選択は自分だけに見える）
    ↓
両者が手を選択すると勝敗判定・コイン移動
    ↓
結果がチャンネルに公開表示される
```

## 賭け金ルール

- 最低賭け金: **10 肩書コイン**
- 最大賭け金: **5,000 肩書コイン**
- 挑戦者・相手ともに賭け金以上の残高が必要です
- 引き分けの場合はコインが返却されます

## 注意

- 自分自身には挑戦できません。
- 既に進行中のじゃんけんがある状態では新たに挑戦できません。
- 挑戦を受けた後に両者が手を選ばないとゲームが完了しません（タイムアウトは5分）。
- 相手が拒否した場合、コインの移動はありません。
- `/economy join` 未参加のユーザーには挑戦できません。
