# タスク管理

サーバー内のタスクを一元管理できる機能です。優先度・期限の設定に対応し、ロールごとに追加権限を付与できます。

## コマンド

```bash
/task add name:<タスク名> [priority:<high|medium|low>] [deadline:<YYYY-MM-DD>]
```

タスクを追加します。優先度はデフォルトで `medium`（中）です。

```bash
/task list
```

全タスクを一覧表示します（全員が閲覧可能・公開表示）。

```bash
/task complete id:<タスクID>
```

指定したタスクを完了済みにします。

```bash
/task delete id:<タスクID>
```

指定したタスクを削除します。

```bash
/task allow-role role:<ロール>
```

指定したロールにタスク追加権限を付与します。

```bash
/task remove-role role:<ロール>
```

指定したロールのタスク追加権限を取り消します。

```bash
/task allowed-roles
```

タスク追加が許可されているロールの一覧を表示します。

## 権限

| サブコマンド | 必要権限 |
|---|---|
| `add` | メッセージの管理 または 許可ロール |
| `list` | なし（全員使用可能） |
| `complete` | サーバーの管理 |
| `delete` | サーバーの管理 |
| `allow-role` | サーバーの管理 |
| `remove-role` | サーバーの管理 |
| `allowed-roles` | サーバーの管理 |

## 優先度アイコン

| 優先度 | アイコン |
|---|---|
| high（高） | 🔴 |
| medium（中） | 🟡 |
| low（低） | 🟢 |

## 注意

- タスクの上限は1サーバーあたり100件です。
- 期限は `YYYY-MM-DD` 形式（例: `2026-12-31`）で入力してください。
- `list` は自分だけでなく全員に公開されます。
