# お祝い保存

メッセージの右クリックメニューから、お祝いや記念のメッセージをアーカイブチャンネルに保存できる機能です。

## コマンド

```bash
/celebration-setup channel:<チャンネル> role:<ロール>
```

お祝い保存機能のセットアップを行います。アーカイブ先チャンネルと、保存を許可するロールを設定します。

**必要権限:** サーバーの管理

---

```
メッセージを右クリック → アプリ → お祝い保存
```

対象メッセージをアーカイブチャンネルに埋め込み（Embed）形式で保存します。

**必要権限:** `/celebration-setup` で設定した許可ロール

## フロー

```
/celebration-setup でアーカイブ先チャンネルと許可ロールを設定
    ↓
保存したいメッセージを右クリック → アプリ → お祝い保存
    ↓
アーカイブチャンネルに Embed 形式で投稿
    （投稿者名・アイコン・本文・元メッセージリンク・保存者名を記録）
```

## 保存内容

アーカイブされる Embed には以下の情報が含まれます：

- 投稿者のアイコンと名前
- メッセージ本文
- 元メッセージへのリンク
- 投稿日時
- 保存者の名前（フッター）
- 添付画像（1枚目は Embed 内に表示、2枚目以降はURLで別途記載）

## 注意

- セットアップ前に「お祝い保存」を使用するとエラーになります。
- テキストのないメッセージは「（テキストなし）」と表示されます。
